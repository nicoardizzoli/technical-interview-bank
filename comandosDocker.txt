Definition: Docker is an open source containerization platform. It enables developers to package applications into containers

- con ctrl + r busco en los comandos que corri

- ver los containers que hay:
    docker ps  //por default muestra solos los que estan UP, si queremos ver todos le agregamos el -a

    docker ps -h //h de help, te muestra una ayuda para ver como se puede combiar el comando.

-para crear un container a partir de la image:
 docker run --name nombreContainer -d -p 80:80 nombreimage     // el -p es port, los ports son->  host:container, el -d es para que corra en background

- para iniciar un conteiner ya creado:
  docker start nombreContainer

- parar un container
    docker stop nombreConteiner

- borrar un container
  docker rm -f nombrecontainer

- FORMATEAR la salida de docker ps (SOLO EN MAC O LINUX) :
export FORMAT="ID\t{{.ID}}\nNAME\t{{.Names}}\nIMAGE\t{{.Image}}\nPORTS\t{{.Ports}}\nCOMMAND\t{{.Command}}\nCREATED\t{{.CreatedAt}}\nSTATUS\t{{.Status}}\n"

luego:
docker ps --format=$FORMAT
